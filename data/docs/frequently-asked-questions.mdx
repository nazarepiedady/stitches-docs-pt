### Quais navegadores a Stitches suporta?

A Stitches funciona em todos navegadores modernos. Edge (Chromium), Chrome, Safari, Firefox - as duas últimas versões, pelo menos.

Especificamente, a Stitches funcionará em navegadores cujos motores de JavaScript suportarem a `Symbol` e `Set`. A experiência do complexo, encaixamento recursivo de estilos só estará visível para os navegadores cujos motores de CSS suportarem `:is()`.

### A Stitches está limitada ao React?

Não. Tu podes utilizar a `@stitches/core` com a Vue, Svelte, Angular, etc. É abstração agnóstica. A principal diferente entre este e o envolvedor de React é que ele retorna as funções de nome da classe, no lugar de componentes.

### Onde eu posso ver a Stitches sendo utilizada em produção?

Estás olhando para ela!

Este sítio foi construído utilizando a Stitches, mais precisamente, utilizando [componentes](https://github.com/radix-ui/design-system) de um sistema de desenho construído utilizando a Stitches.
