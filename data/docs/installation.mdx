### Instalar a Stitches

Instale a Stitches a partir do teu terminal via npm ou yarn.

```bash
# Com o npm
npm install @stitches/react

# Com o yarn
yarn add @stitches/react
```

### Importe-a

Importe `styled` de `@stitches/react`.

```js
import { styled } from '@stitches/react';
```

### Utilize-a

Utilize a função `styled` para criar um componente e adicionar estilos à ele.

```jsx line=3-11
import { styled } from '@stitches/react';

const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  padding: '10px 15px',
  '&:hover': {
    backgroundColor: 'lightgray',
  },
});
```

### Interprete-a

Finalmente, interprete o componente.

</Preview>

```jsx line=5
import { styled } from '@stitches/react';

const Button = styled('button', {...});

() => <Button>Button</Button>;
```

### Funções disponíveis

Além de `styled`, estas são as outras funções disponíveis:

```jsx
import {
  styled,
  __css__,
  __globalCss__,
  __keyframes__,
  __theme__,
  __createTheme__,
  __getCssText__,
} from '@stitches/react';
```

Consulte a [página da API](/docs/) para aprender mais sobre cada uma delas.

## Configurar a Stitches

### Criar o teu ficheiro de configuração

Para configurar a Stitches, crie um ficheiro `stitches.config.ts` (`.js` também funciona) e importe a função `createStitches`.

```jsx
// stitches.config.ts
import { createStitches } from '@stitches/react';
```

Esta função recebe um objeto de configuração:

- `theme`: Define o teu [tema](/docs/tokens) de desenho, que delineia para propriedades de CSS.
- `media`: Define [pontos de quebra](/docs/breakpoints) responsivos e reutilizáveis.
- `utils`: Cria [utilitários](/docs/utils) personalizados para melhorar tua experiência de programação.
- `prefix`: Prefixa nomes de classe e variáveis de CSS para evitar colisões. [Aprenda mais](/docs/api#prefix).
- `themeMap`: Define um delineamento personalizado de propriedades de CSS para símbolos de tema. [Aprenda mais](/docs/api#thememap).

E retorna todas as [funções disponíveis](/docs/installation.mdx#funções-disponíveis) acima.

```js
// stitches.config.ts
import { createStitches } from '@stitches/react';

export const {
  styled,
  css,
  globalCss,
  keyframes,
  getCssText,
  theme,
  createTheme,
  config,
} = createStitches({
  theme: {
    colors: {
      gray400: 'gainsboro',
      gray500: 'lightgray',
    },
  },
  media: {
    bp1: '(min-width: 480px)',
  },
  utils: {
    marginX: (value) => ({ marginLeft: value, marginRight: value }),
  },
});
```

> Para ver todas opções de configuração, consulte a [página da API](/docs/api#createStitches).

### Importe e utilize-a

A partir deste ponto em diante, estarás importando `styled` e outras funções de <H id="one" index="1">stitches.config</H>.

## Utilizando marcadores de `script`

Se preferires, podes adicionar a Stitches ao teu sítio utilizando marcadores de `script`.

```html
<script src="https://unpkg.com/@stitches/react" crossorigin></script>
```

Depois utilize-a desta maneira:

```jsx
// sem configuração
const { styled, css, globalCss } = stitches;

// com configuração
const { styled } = stitches.createStitches({ ...config });
```
