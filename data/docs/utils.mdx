---
title: Utilitários
description: Crie utilitários personalizados para uma experiência de programação mais agradável.
---

Crie as tuas próprias propriedades de CSS no objeto `utils`. Adicione quantos utilitários precisares. Abaixo está alguns exemplos:

```jsx
export const { styled, css } = createStitches({
  utils: {
    // Propriedades de margens abreviadas
    m: (value) => ({
      margin: value,
    }),
    mt: (value) => ({
      marginTop: value,
    }),
    mr: (value) => ({
      marginRight: value,
    }),
    mb: (value) => ({
      marginBottom: value,
    }),
    ml: (value) => ({
      marginLeft: value,
    }),
    mx: (value) => ({
      marginLeft: value,
      marginRight: value,
    }),
    my: (value) => ({
      marginTop: value,
      marginBottom: value,
    }),

    // Um propriedade para aplicação em conjunto de `width` e de `height`
    size: (value) => ({
      width: value,
      height: value,
    }),

    // Uma propriedade aplicar gradiente linear
    linearGradient: (value) => ({
      backgroundImage: `linear-gradient(${value})`,
    }),

    // Um propriedade abreviada para `border-radius`
    br: (value) => ({
      borderRadius: value,
    }),
  },
});
```

> `value` será o valor passado através da propriedade de CSS.
