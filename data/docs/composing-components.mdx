---
title: Composição de Componentes
description: Compondo um componente de Stitches para um outro.
---

Os componentes de Stitches podem ser compostos através da função `styled`.

<Preview>
  <DemoButton children="Base button" css={{ mr: '$4' }} />
  <DemoButton
    children="Checkout button"
    css={{
      borderRadius: 0,
      backgroundColor: '$pink10',
      color: 'white',
      '&:hover': {
        backgroundColor: '$pink11',
      },
    }}
  />
</Preview>

```jsx
const BaseButton = styled('button', {});

const CheckoutButton = styled(BaseButton, {
  borderRadius: 0,
  backgroundColor: 'hotpink',
  color: 'white',
  '&:hover': {
    backgroundColor: 'deeppink',
  },
});

() => (
  <>
    <BaseButton>Base button</BaseButton>
    <CheckoutButton>Checkout button</CheckoutButton>
  </>
);
```

Quando compões um componente que possui variantes, elas serão herdadas.

```jsx line=4-9,12-14,16-23
const BaseButton = styled('button', {
  // estilos de base

  variants: {
    size: {
      small: {},
      large: {},
    },
  },
});

const CheckoutButton = styled(BaseButton, {
  // estilos do `checkout`
});

() => (
  <>
    <BaseButton size="small">Base button</BaseButton>
    <CheckoutButton size="small">Checkout button</CheckoutButton>
    <BaseButton size="large">Base button</BaseButton>
    <CheckoutButton size="large">Checkout button</CheckoutButton>
  </>
);
```
